Parameters

NewtonKrylov.Forcing — Type
Forcing

Implements forcing for inexact Newton-Krylov. The equation $‖F′(u)d + F(u)‖ <= η * ‖F(u)‖$ gives the inexact Newton termination criterion.

Implemented variants

  • Fixed
  • EisenstatWalker
